[베리온대부 홈페이지 설정]

1. firebase-config.js에 Firebase 프로젝트 설정값을 입력합니다.
2. Firebase Authentication에서 관리자 이메일 계정을 생성합니다.
3. firestore.rules를 Firebase Firestore 규칙에 배포합니다.
4. 모든 파일을 GitHub 저장소 루트에 업로드합니다.
5. admin.html에서 신청현황, 읽음, 상담완료, 삭제 및 실시간상담을 관리합니다.
6. 배포 전 개인정보처리방침, 대출상품 금리/한도/기간, 광고심의필 번호 및 유효기간을 실제 승인 내용에 맞게 수정합니다.
7. 랜딩페이지를 포함한 온라인 광고물은 한국대부금융협회 광고심의가 필요할 수 있습니다.

관리자 주소: https://도메인/admin.html
